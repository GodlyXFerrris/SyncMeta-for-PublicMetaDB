Quick Setup
1
Enter a profile password. Leave UUID blank for a new profile. Click Save Profile.
2
Add your PublicMetaDB API key in Connections. Connect SIMKL, Trakt, AniList, or MDBList.
3
Pick lists in Lists. Return to Dashboard and click Sync Lists.
Use this UUID + password to reopen the same profile in another browser or server.
Use your profile UUID plus a new password to reset access.

List Sync
Schedule and behavior for syncing lists from your connected sources into PublicMetaDB.
Minimum 21600 s (6 h). Default 43200 s (12 h). Scheduled from end of last successful run.

Watch History
Import watched history into PublicMetaDB. The selected source is the only write direction for this rule.
Minimum 120 s (24 h). Only affects automatic history sync.
Resume Progress
Import continue-watching progress. When background sync is on, this refreshes automatically on its own schedule.
Minimum 120 s (24 h). Only affects automatic resume sync.
